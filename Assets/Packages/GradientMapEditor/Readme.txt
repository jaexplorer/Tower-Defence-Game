[Gradient map tool by Vladislav Kostin]

Create a window of this editor by selecting [Window]>[Gradient Map Editor]
Create a GradientMap asset [Create]>[Data]>[GradientMap] //You can change the path in the GradientMap file.
You must create assign the texture to the new GradientMap manually in inspector.
	Texture must have Read/Write enabled, and have no compression.
When GradientMap editor is open to edit a GradientMap select GradientMap asset in Project view or press the button on top of the GradientMap editor and select from the list.
